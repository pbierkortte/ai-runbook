# Obsidian CLI Help

Obsidian CLI lets you control Obsidian from your terminal.

The user wants to drive Obsidian from the shell, and you don't have the current CLI syntax memorized.

Required Steps:
1. Run `obsidian help` (or `obsidian help <command>` for a specific subcommand)
2. Run the appropriate `obsidian <subcommand>`

You now hold the current CLI surface in context and can act precisely.

Applies to any `obsidian` subcommand for the rest of this session.
